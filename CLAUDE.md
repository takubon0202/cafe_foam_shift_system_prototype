# プロジェクト概要

このプロジェクトは、サブエージェントシステムを活用したWebサイト開発プロジェクトです。

## サブエージェント構成

各サブエージェントは独立した役割を持ち、順番にタスクを引き継いで開発を進めます。

### ワークフロー

```
[要件定義] → [デザイン] → [コーディング] → [バックエンド] → [GASセットアップ] → [画像生成] → [デバッグ] → [ペルソナ評価] → [評価者] → [改善ループ]
```

### サブエージェント一覧

| コマンド | 役割 | 説明 |
|---------|------|------|
| `/workflow` | ワークフロー管理 | 全体の開発フローを統括・実行 |
| `/design` | デザイン担当 | UI/UXデザイン、カラースキーム設計 |
| `/coding` | コーディング担当 | HTML/CSS/JSの実装 |
| `/backend` | バックエンド担当 | GAS・サーバーサイド・API実装 |
| `/gas-setup` | GASセットアップ | スプレッドシート・GAS連携ガイド |
| `/image-gen` | 画像生成担当 | 必要な画像の生成プロンプト作成 |
| `/debug` | デバッグ担当 | バグ修正、動作確認 |
| `/responsive` | レスポンシブ担当 | iOS/Android/iPad対応・クロスブラウザ |
| `/persona` | ペルソナ評価 | ターゲットユーザー視点での評価 |
| `/evaluator` | 評価者 | SEO・事業観点での総合評価 |

## GAS連携クイックリファレンス

### セットアップ手順（10分）
1. Google スプレッドシート新規作成
2. 拡張機能 → Apps Script
3. `cafe-unified-system/gas/Code.gs` を貼り付け
4. `initializeSheets` を実行（権限許可）
5. デプロイ → ウェブアプリ → 全員
6. URLを `js/config.js` に設定

### API一覧
| アクション | 説明 |
|-----------|------|
| submitShifts | シフト提出（週1制約） |
| deleteShift | シフト削除 |
| punch | 打刻（遅刻/早退判定） |
| getShifts | シフト取得 |
| getClockRecords | 打刻記録取得 |

## デザインガイドライン

### 禁止事項
- 絵文字の使用
- 紫色を基調としたデザイン
- AIっぽいグラデーション（紫→青、ネオン系）
- 過度なアニメーション
- ジェネリックなストック画像風デザイン

### 推奨事項
- 事業内容に合った色彩設計
- オリジナリティのあるレイアウト
- 合理的で使いやすいUI
- 明確な情報階層
- アクセシビリティへの配慮

## 評価基準

### ペルソナ評価（/persona）
- 共感度（10点満点）
- 通いたい度（10点満点）
- サービス利用意欲（10点満点）
- **合格ライン: 各項目7点以上**

### 評価者評価（/evaluator）
- SEO対策（10点満点）
- 事業内容の明確さ（10点満点）
- 分かりやすさ（10点満点）
- 申込意欲喚起度（10点満点）
- **合格ライン: 各項目7点以上**
