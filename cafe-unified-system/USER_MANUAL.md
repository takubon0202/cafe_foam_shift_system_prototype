### 画面の種類

| 画面 | 説明 |
|------|------|
| **シフト提出** | 自分のシフトを登録する画面 |
| **打刻** | 出勤・退勤を記録する画面 |

| 枠名 | 時間 | 説明 |
|------|------|------|
| 午前A | 10:00〜11:30 | 90分 |
| 午前B | 11:00〜12:30 | 90分 |
| 午後A | 15:00〜16:30 | 90分 |
| 午後B | 15:30〜17:00 | 90分 |

- 土日は休みです

## シフトを提出する

### ステップ1: 自分の名前を選ぶ

1. 画面上部のナビゲーションから「**シフト提出**」をタップ
2. 「名前で検索」に自分の名前を入力するか、一覧から選択
3. 選択すると「○○さんを選択中」と表示されます

### ステップ2: シフト枠を選ぶ

1. 各週のカードが表示されます（例：1/12週、1/19週、1/26週）
2. 希望する日の希望する枠（午前A/B、午後A/B）をタップ
3. **1週間につき1枠だけ**選べます

#### 枠の見方

| 表示 | 意味 |
|------|------|
| ○ | 空きあり（0〜1名） |
| △ | あと少し（2名） |
| ● | 満員（3名以上） |
| 自分 | 自分が既に登録済み |

### ステップ3: 提出する

1. 選択が終わったら「**シフトを提出する**」ボタンをタップ
2. 「シフトを提出しました」と表示されたら完了です

### シフトをキャンセルしたいとき

1. 画面下部の「提出済みシフト」を確認
2. キャンセルしたいシフトの「**キャンセル**」ボタンをタップ
3. 確認ダイアログで「**キャンセルする**」をタップ
4. キャンセル後、別の枠を選び直せます

## 打刻する（出勤・退勤）

### 出勤するとき

1. 画面上部のナビゲーションから「**打刻**」をタップ
2. 自分の名前を選択
3. 本日のシフトが表示されます
4. 打刻する枠（午前A/B、午後A/B）をタップして選択
5. 「**出勤**」ボタンをタップ
6. 「出勤を記録しました」と表示されたら完了です

### 退勤するとき

1. 「**打刻**」画面を開く
2. 自分の名前を選択
3. 打刻する枠を選択
4. 「**退勤**」ボタンをタップ
5. 「退勤を記録しました」と表示されたら完了です

### 打刻のルール

| ルール | 説明 |
|--------|------|
| 出勤可能時間 | シフト開始の**10分前**から打刻できます |
| 遅刻 | シフト開始から**5分以上**遅れると「遅刻」として記録されます |
| 早退 | シフト終了の**15分以上前**に退勤すると「早退」として記録されます |

### 打刻履歴の確認

画面下部の「今日の打刻履歴」で、本日の出勤・退勤時刻を確認できます。

## カレンダーでシフトを確認する

### 全員のシフトを見る

1. 画面上部のナビゲーションから「**カレンダー**」をタップ
2. 月間カレンダーが表示されます
3. 各日付をタップすると、その日のシフト詳細が見られます

### 自分のシフトだけ見る

1. 「スタッフを選択」から自分の名前を選ぶ
2. 自分のシフトがハイライト表示されます

### カレンダーの見方

| 色 | 意味 |
|----|------|
| 緑の背景 | 十分な人数がいる |
| 黄色の背景 | 人が足りない枠がある |
| 青の丸 | 自分のシフト |
| グレー | 営業日ではない |

### 統計情報

画面下部に以下の統計が表示されます：
- 総枠数
- 申請済み
- 人員不足
- 自分の枠

## よくある質問（FAQ）

### Q: 同じ週に2つシフトを入れたい

**A:** 申し訳ありませんが、**1週間に1枠（90分）**のみとなっています。これはルールですので変更できません。

---

### Q: シフトを変更したい

**A:** 以下の手順で変更してください：

1. 「シフト提出」画面を開く
2. 自分の名前を選択
3. 「提出済みシフト」から該当シフトを「キャンセル」
4. 新しい枠を選んで再提出

---

### Q: 出勤ボタンが押せない

**A:** 以下を確認してください：

1. **自分の名前を選択しましたか？**
2. **本日シフトが入っていますか？**
3. **打刻する枠を選択しましたか？**
4. **シフト開始10分前になりましたか？**

---

### Q: 退勤ボタンが押せない

**A:** 先に「出勤」を押す必要があります。出勤していない状態では退勤できません。

---

### Q: 打刻を間違えた

**A:** 管理者（山﨑さん）に連絡してください。手動で修正できます。

---

### Q: 自分の名前がリストにない

**A:** 管理者（山﨑さん）に連絡してください。スタッフリストに追加してもらえます。

---

### Q: 画面が正しく表示されない

**A:** 以下を試してください：

1. ページを再読み込み（スマホ：画面を下に引っ張る / PC：F5キー）
2. ブラウザのキャッシュをクリア
3. 別のブラウザで試す（Chrome推奨）

---

### Q: 午前Aと午前Bの違いは？

**A:** 開始時間が違うだけです：
- **午前A**: 10:00〜11:30
- **午前B**: 11:00〜12:30

どちらも90分で、やることは同じです。都合の良い方を選んでください。

---

### Q: 午後Aと午後Bの違いは？

**A:** 開始時間が違うだけです：
- **午後A**: 15:00〜16:30
- **午後B**: 15:30〜17:00

どちらも90分で、やることは同じです。都合の良い方を選んでください。

## 困ったときは

### 管理者への連絡

システムの問題や不明点は、管理者の**山﨑 琢己**さんに連絡してください。

### よくあるエラーメッセージ

| エラー | 意味 | 対処法 |
|--------|------|--------|
| 「この週には既にシフトが登録されています」 | 同じ週に2つ目を登録しようとした | 既存のシフトをキャンセルしてから再登録 |
| 「このシフト枠はすでに出勤済みです」 | 同じ枠で2回出勤しようとした | 既に出勤しているので何もしなくてOK |
| 「先に出勤打刻をしてください」 | 出勤せずに退勤しようとした | 先に「出勤」ボタンを押す |
| 「スタッフを選択してください」 | 名前を選んでいない | ドロップダウンから自分の名前を選択 |

## 操作の流れ（まとめ）

### シフト提出の流れ

```
1. サイトにアクセス
   ↓
2. 自分の名前を選択
   ↓
3. 各週で1枠を選択
   ↓
4. 「シフトを提出する」をタップ
   ↓
   完了！
```

```
【出勤時】
1. 「打刻」画面を開く
   ↓
2. 自分の名前を選択
   ↓
3. シフト枠を選択
   ↓
4. 「出勤」をタップ
   ↓
   出勤完了！

【退勤時】
1. 「打刻」画面を開く
   ↓
2. 自分の名前を選択
   ↓
3. シフト枠を選択
   ↓
4. 「退勤」をタップ
   ↓
   退勤完了！
```
